Angular JS           vs.        Angular
-----------                    ---------

* Library                      Framework
* v1.x                         v2+

create angular app : ng new name

to create with module : ng new name --standalone

run : ng serve

create a component : ng g c parent

create component in parent folder : ng g c parent/child
